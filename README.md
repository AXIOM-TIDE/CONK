## Seven Primitives. Three Laws. One Mission.

## The Primitives
```
1  HARBOR       The wallet. The fuel. Never sees a cast.
2  VESSEL       The identity. Mortal by design.
3  CAST         The communication primitive. Everything is a cast.
4  DRIFT        The public feed. The tide votes here.
5  DOCK         The sealed room. Private casts only.
6  SIREN        The open broadcast. Pulls vessels to a Dock.
7  LIGHTHOUSE   The permanent record. Earned by the tide.
```

## The Three Laws
```
I.   Casts never reach the Harbor. Ever.
II.  The Harbor knows only that balance decreased.
III. The connection between who paid and what was cast
     does not exist in this protocol.
     Because it was never built.
```
